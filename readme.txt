淘宝客

== Description ==

在Wordpress博客系统中显示带有属于自已的淘宝客链接的商品页面

演示地址:[https://www.aliluv.cn/shop](https://www.aliluv.cn/shop)

插件设置帮助：[https://www.aliluv.cn/18.html](https://www.aliluv.cn/18.html)

== Installation ==

1. 上传 `nines-taoke`目录 到 `/wp-content/plugins/` 目录或后台搜索Nines Taoke
2. 在后台插件菜单激活该插件
3. 后台`淘宝客配置`填写你的 `App Key` ,`App Secret` ,`Pid` 三个配置项以及新建一个新页面用来展示淘客商品
4. 其中pid参数必须是淘宝联盟->网站推广位  的pid(没有这个推广位的pid请申请),其它位置的pid将获取不到商品数据、详细教程请点击 `插件设置帮助`


== Frequently asked questions ==

如果在页面中显示(不是商品数据时)错误信息时:多半是App Key或者App Secret的原因,自行检查;
提示 “未能找到相关商品数据” 多半是Pid不属于 网站推广位 ;
